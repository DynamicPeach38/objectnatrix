
### HOW TO
**Steps to Success**

**1)** cd to Github/ 
**2)** touch *new file.ex* 
**3)** type `git status` 
**4)** type `git add new file.ex` 
**5)** type `git status` (Make sure all is green!) 
**6)** type `git push` - enter your login information if you haven't already
**7)** Done! Your file is on the Github! hooray!
